# sing-box
自制 sing-box 工具箱。支持的功能：

1. Install sing-box
2. Upgrade sing-box
3. Uninstall sing-box
4. Start sing-box
5. Stop sing-box
6. Restart sing-box
7. Check status
8. Show config
9. Reset config


## 一键脚本

```bash
bash <(curl -s https://raw.githubusercontent.com/sunnypuppy/sing-box/main/singbox_tools.sh)
```
